# Power BI Financial Analysis Dashboard

## Project Overview
This project is a multi-page Power BI dashboard built to analyze multi-year financial data and evaluate an organizationâ€™s financial health. The dashboard tracks key performance indicators, identifies trends, and supports data-driven financial decision-making.

The analysis covers multiple years of financial data and focuses on revenue performance, profitability, and overall financial trends using interactive visuals.

## Key Objectives
- Analyze financial performance across multiple years
- Track key financial KPIs and metrics
- Identify trends and patterns in financial data
- Provide an interactive dashboard for business insights

## Key Features
- Multi-page Power BI dashboard
- KPI cards for revenue and performance metrics
- Year-over-Year trend analysis
- Interactive slicers and filters
- Clean data modeling and DAX calculations

## Tools & Technologies
- Power BI
- Power Query
- DAX
- Excel / CSV datasets

## Dataset Information
- Financial data covering multiple years
- Dataset used for analytical and educational purposes
